This plugin give you the Bootstrap Carousel with powerfull ability to make it and define id ,time ,drag and drop ability to order the silder and much more ...

== Description ==

* Build for Booststrap Carousel with Wordpress ;I did that for my needs 
* Simple Carousel with caption and without 
* Simple interface to upload and order the slides.
In this plugin you will able to upload your files in seperate folder (Not the Default - uploads) thats will keep your files safe 
* Ease to remove and drag and drop ability to order the sildes as you want 
* You can define an id for you carousel to style it as you want and an option for the time interval 
* Simple shortcode will shown in the carousels list will show ,copy and past it where you need 
** this plugin can makes as much as you want of slider not like others make one and only .


hope you like it :)


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add your Slider
4. Upload sildes in it and add the caption you want.
5. Shortcode will generated for you in the sliders list.
6. thats it . nothing complicated
